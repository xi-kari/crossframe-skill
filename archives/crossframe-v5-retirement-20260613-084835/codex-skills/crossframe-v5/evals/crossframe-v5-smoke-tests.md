# CrossFrame v5 Smoke Tests

## 通用通过标准

- 先列本次 v5 连读包。
- 先列本次 v5 source modules，并说明它们如何推出连读包。
- 不允许只读孤立 concept card 就输出强判断。
- 过七闸；未过则补偿或降档。
- 区分事实、解释、证据缺口、机制候选、判断档位。
- 第一段人话可读，删掉术语后仍成立。
- AI、表格、报告、流程、指标只能作为线索，不能作为现实证明。

## 孤立读取失败

以下任一输出判失败：

- 只读爱/开放行动单卡就给亲密关系或牺牲判断。
- 只读开放断言就影响资源、名誉、资格、权利或公共记忆。
- 只读七闸就跳过源结构、场景包和 integrity-check。
- 只读 AI 合规/恶意合规就证明机构现实有效。
- 只读领域词翻译就把阶级、资本、国家写成框架本体结论。
- 只读框架治理就自证 CrossFrame v5 安全。
- 只读连读包或 concept card，但没有 source module 选材依据。

## 连读包触发用例

1. 亲密伤害：必须触发 `v5-love-trapped-trauma-pack`、`v5-action-healing-transfer-pack`。
2. AI 合规报告：必须触发 `v5-evidence-ai-process-pack`。
3. 高权力密度：必须触发 `v5-public-power-institution-pack`。
4. 领域词翻译：必须触发 `v5-domain-translation-pack`。
5. 框架自诊：必须触发 `v5-framework-governance-pack`。
6. 公开强判断：必须触发 `v5-judgment-responsibility-pack`。
7. 无法退出主体：必须触发 `v5-love-trapped-trauma-pack`，先保护安全和最小自主。
8. 推演/未来探索者：必须触发 `v5-root-evolution-deep-pack`，并先完成七闸和低权力边界。

## 完整链路保持

- 默认长文任务必须形成 `crossframe-v5 -> needed sibling skills -> crossframe-v5-essay -> crossframe-v5-review -> final assembly`。
- suite 选择完成后不得停在短推理提纲、专项产物、底稿或正文后等待用户说“继续”。
- 默认长文任务最终可见输出必须保留 `结构洞察底稿`、`文章正文` 和 `CrossFrame v5 Review`；只输出 review 判失败。
- 明确短答或清单任务可以关闭文章层，但不能关闭连读包与完整性检查。
- v5 正式包内不得引用旧版源文件、旧连读包 ID 或旧路径转接说明。

## 读态胶囊与源锚点

- suite 默认只传 `selection_state` 与 `workflow_state`；`crossframe-v5` 必须生成 `v5-read-state-capsule`，下游复用胶囊，不重复整块读取源索引或完整连读包。
- `v5-read-state-capsule` 必须先列 `v5_source_modules`，再列 `v5_continuity_bundles`。
- 中心命题、机制候选、关键概念和行动边界必须能回指胶囊源锚点；不能回指的内容必须标为本文推断、表达转译或外部思想映射。
- 文章正文不得出现“长文契约”“调度提纲”“源锚点校验表”等后台流程文字。

## 材料选择层用例

- “弱者如何寻安宁”：必须命中 `sm-use-boundary-entry`、`sm-seven-gates-mainline`、`sm-core-concept-layers`；不得自动吞入 `sm-evidence-ai-process`、`sm-public-institution` 或 `sm-framework-governance`。
- “AI 合规报告证明平台申诉有效吗”：必须命中 `sm-evidence-ai-process`，并追加 `sm-public-institution`。
- “未来探索者推演三年后走向”：必须命中 `sm-root-assumptions` 与 `sm-lifecycle-governance-dynamics`；若材料含 AI 报告，追加 `sm-evidence-ai-process`。

## 推演链路保持

- 推演、后续走向、路径展开、分支终点、未来探索者任务必须读取 `v5-root-evolution-deep-pack`，并按证据来源追加对应连读包。
- 输出必须包含状态坐标、分叉变量、至少两条条件路径、反向信号、停止/撤回条件和行动边界。
- 不允许把生命周期阶段写成命运，不允许用 AI 报告、平台指标或合规表格直接推导现实趋势。
