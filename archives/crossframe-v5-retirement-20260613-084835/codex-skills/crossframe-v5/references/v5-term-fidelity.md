# v5 Term Fidelity

本文件记录 v5 高风险术语的最低保真要求。它不替代概念卡；当术语承担判断作用时，还必须读取所属连读包。

| 术语 | v5 保真要求 | 不可误读 |
| --- | --- | --- |
| 七闸 | G1-G7 是实质诊断硬前置，先决定判断能不能说，再决定内容。 | 不是普通 checklist；未通过不得进入强判断或行动建议。 |
| 低权力主体 | 缺乏退出、申诉、信息、替代资源或安全表达条件的位置。 | 不得要求其为对话、大局、爱或修复继续暴露。 |
| 开放断言 | 低到中置信度的可撤回判断，必须带证据、替代解释、撤回条件和行动边界。 | 不是处置依据；被写入档案/资源分配时必须退场。 |
| 过程性产物 | AI 文本、报告、流程、表格、脚本通过只证明过程材料存在。 | 不能当作现实证明或强证据。 |
| 观测参与 | 诊断会改变对象，必须区分基线、观测中反应、发布后反应。 | 不能把被诊断激发的反应当作对象本质。 |
| 高权力密度 | 关键通道被少数节点耦合控制时，末端反馈和申诉反噬权重上调。 | 不能把中心叙事或名义程序当有效程序。 |
| 领域词 | 强领域词先翻译为结构变量，再进入判断。 | 阶级、资本、国家等不能直接升格为框架本体。 |
| 爱 | 作为规范前提和开放性承担行动的生成事件位置，诊断层只观察成本、回流、保护、继承、停止权。 | 不能神化，不能命令，不能当正当性豁免。 |
| 无法退出 | 优先安全、代理保护、最小自主、低暴露记录和替代出口。 | 不能要求理想退出，不能解释为忠诚或成熟。 |
| 框架自诊 | 区分文本、概念、实践、社群；反例必须产生文本后果。 | 不能用 CrossFrame v5 自证 CrossFrame v5 安全。 |
| 良性消亡 | 当外部方法更低误伤、更有效，框架应降级、转接或退场。 | 成熟不是解释一切，而是知道何时不用自己。 |
