# 公共评论底稿模板

## 证据边界

- 本文可依赖的事实：
- 只能作为弱信号的材料：
- 平台/机构声明的证据档位：
- 仍需核验的问题：

## 中心命题

一句话写清评论要批评的不是人格，而是哪个公共机制失灵。

## 正文草稿

1. 具体入口：从一个可核验事实或程序矛盾进入。
2. 机制展开：说明规则、执行、申诉、弱信号或承诺偿付如何断裂。
3. 责任边界：指出谁有改变条件，谁在承担成本。
4. 反向条件：写明什么证据会改变判断。
5. 行动收束：提出可复核、可撤回、不过度伤害的公共要求。

## 结尾姿态

结尾不喊口号，不做人格审判；把问题收回到制度纠错和公共承诺偿付。
