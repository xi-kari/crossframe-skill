# 众星捧月法

- 定义：用多个材料、道理或侧面共同托出同一个中心主题。
- 适用文章类型：中性分析长文、思想/概念阐释文章、读书互读/吸收文章。
- 解决的问题：一个例子不足以支撑论点，但多例又容易散。
- 操作步骤：先确定“月”即中心命题；每个材料只承担一个侧面；段末回扣中心。
- 原书操作要点（转述）：原书把这类方法放在“形与神”的关系中理解：先让中心意思立住，再让材料、结构和语言围绕它成形。本技法的关键不是记住名称，而是把“用多个材料、道理或侧面共同托出同一个中心主题”落实到材料选择和段落位置上；执行时按“先确定“月”即中心命题；每个材料只承担一个侧面；段末回扣中心”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“一个例子不足以支撑论点，但多例又容易散”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p19；OCR 对应 PDF_PAGE 028 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：拱主题句：每个材料段都用一句短句把“星”托回“月”。
- 段落前后关系：前接中心命题、典型材料或关键细节；使用它先确立精神焦点。后接事实展开、机制解释或结尾收束，确保所有材料继续围绕主心骨。
- 文章类型微用法：答复体用它抓住读者最在意的一句话；评论体用它确立主心骨；思想文章用它把抽象概念压回可记忆的精神焦点；案例文用它防止材料散开。
- 失败示例（转述）：失败时会把漂亮中心句贴到材料外面，读者只记住口号，却看不见事实、机制和边界。
- 文章类型用法：适用于中性分析长文、思想/概念阐释文章、读书互读/吸收文章。优先用于主心骨、首段入口、中心句或结尾收束，让全文有一个稳定的精神焦点。
- 失败形态：材料不能喧宾夺主，不能互相重复；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：一字立骨法、分层论说法、古今中外法。
- 误用风险：材料不能喧宾夺主，不能互相重复。
- 输出自检：每个“星”是否都从不同角度托出同一个“月”？
