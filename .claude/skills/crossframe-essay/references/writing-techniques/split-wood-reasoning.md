# 析薪破理法

- 定义：按事物内在纹理层层剖析，把复杂问题讲透。
- 适用文章类型：思想/概念阐释文章、中性分析长文、组织复盘文章。
- 解决的问题：道理混在一起，读者无法判断层次。
- 操作步骤：找出问题内部结构；按因果、责任或尺度逐层展开；每层只解决一个问题。
- 原书操作要点（转述）：原书强调议理要顺着问题的内在纹理推进：论据、层次、反证和边界要共同服务于把道理讲透。本技法的关键不是记住名称，而是把“按事物内在纹理层层剖析，把复杂问题讲透”落实到材料选择和段落位置上；执行时按“找出问题内部结构；按因果、责任或尺度逐层展开；每层只解决一个问题”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“道理混在一起，读者无法判断层次”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p270；OCR 对应 PDF_PAGE 279 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：剖理句：顺着事理纹路一层层劈开，不跳步。
- 段落前后关系：前接待辨析命题、反方论据或机制候选；使用它安排论证动作。后接证据边界、反向条件和撤回条件，防止论证滑向动员口号。
- 文章类型微用法：论辩文章用它拆命题和反驳弱论据；公共评论用它建立责任链；思想阐释用它讲透概念关系；组织复盘用它把责任、授权和反馈分层。
- 失败示例（转述）：失败时会把论证锋利度误当事实强度，省掉来源台账、反方条件和撤回路径。
- 文章类型用法：适用于思想/概念阐释文章、中性分析长文、组织复盘文章。优先用于论证段、批判段、反驳段和撤回条件，让观点有证据、有层次、有反向压力。
- 失败形态：不能半途而止，也不能随意切割；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：分层论说法、釜底抽薪法、类比说理法。
- 误用风险：不能半途而止，也不能随意切割。
- 输出自检：每一层是否都沿着真实纹理推进？
