# 疏笔勾勒法

- 定义：用简练笔触抓住对象本质特征，少写但写准。
- 适用文章类型：答复体文章、公共评论文章、中性分析长文。
- 解决的问题：描写臃肿，正文节奏拖慢。
- 操作步骤：找最能显出对象的两三处特征；用简短句写出；立刻转入判断。
- 原书操作要点（转述）：原书把这类方法看作材料进入表达的秩序：叙述、描写、议论和抒情要随对象变化，而不是平均铺开。本技法的关键不是记住名称，而是把“用简练笔触抓住对象本质特征，少写但写准”落实到材料选择和段落位置上；执行时按“找最能显出对象的两三处特征；用简短句写出；立刻转入判断”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“描写臃肿，正文节奏拖慢”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p94；OCR 对应 PDF_PAGE 103 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：白描句：少形容、多特征，用最少笔触点出对象骨架。
- 段落前后关系：前接已筛过的事实、场景或用户材料；使用它决定材料怎样被读者看见。后接议论、机制判断或情绪回落，避免表述段变成孤立描写。
- 文章类型微用法：答复体用它把处境说清；公共评论用它交代事实和现场；案例文用它组织人物与时间线；中性分析长文用它降低术语密度。
- 失败示例（转述）：失败时会沉迷描写、节奏或现场感，让材料变成气氛，削弱事实边界和判断档位。
- 文章类型用法：适用于答复体文章、公共评论文章、中性分析长文。优先用于段落组织、叙述顺序、视角转换或语气节奏，让材料进入可读的表达秩序。
- 失败形态：简不是粗，必须抓准本质；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：以点传神法、点面结合法、意在言外法。
- 误用风险：简不是粗，必须抓准本质。
- 输出自检：删掉多余形容后，特征是否仍清楚？
