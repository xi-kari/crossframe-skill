# 点面结合法

- 定义：既写整体面貌，也抓重点局部。
- 适用文章类型：组织复盘文章、案例分析文章、中性分析长文。
- 解决的问题：只有宏观会空，只有局部会窄。
- 操作步骤：先交代“面”；选出关键“点”；用点验证或修正面对问题的理解。
- 原书操作要点（转述）：原书把这类方法放在点面、虚实、动静、显隐之间理解：对照和转化要让主体更清楚，而不是遮蔽主体。本技法的关键不是记住名称，而是把“既写整体面貌，也抓重点局部”落实到材料选择和段落位置上；执行时按“先交代“面”；选出关键“点”；用点验证或修正面对问题的理解”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“只有宏观会空，只有局部会窄”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。

- 好句类型：点面切换句：从局部证据跳到整体机制，再回到局部限制。
- 段落前后关系：前接局部材料或整体背景；使用它处理点面、虚实、多寡、动静关系。后接边界段、机制段或概念上升，保持对照和转化有现实落点。
- 文章类型微用法：中性分析长文用它处理复杂材料；公共评论用它兼顾局部案例和制度面；思想文章用它保留余味；组织复盘用它避免单线归因。 适合公共、组织、案例和中性分析；点面切换时要写清从局部能推到哪里、不能推到哪里。
- 失败示例（转述）：失败时会把对照、留白或象征写得含混，导致中心命题和责任链消失。
- 文章类型用法：适用于组织复盘文章、案例分析文章、中性分析长文。优先用于复杂关系、点面互照、含蓄表达或边界段，让判断保留层次和余地。
- 失败形态：面不能空泛，点不能孤立；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：以点传神法、以少胜多法、彩线穿珠法。
- 误用风险：面不能空泛，点不能孤立。
- 输出自检：点和面之间是否互相说明？
