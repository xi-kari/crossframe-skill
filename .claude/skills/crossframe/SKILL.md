# CrossFrame for Claude Code

本 Claude Code skill 是薄适配层。权威主体在 `skills/crossframe/SKILL.md`。

使用时必须：

1. 读取 `skills/crossframe/SKILL.md`。
2. 读取 `skills/crossframe/references/read-routing-map.md`，根据任务类型选择 protocol、worksheet、concept card 和模板。
3. 对高责任、公共制度、亲密关系、长期演化、深度分析和文章输出，读取 `skills/crossframe/references/continuity-bundles.md`，必要时读取 `skills/crossframe/references/v3-source-spine.md` 和 `skills/crossframe/references/v3-section-digest-index.md`。
4. 默认先输出可见推理提纲。
5. 先说人话，再按需补术语映射。
6. 不把结构诊断写成人格审判、宿命判断或概念堆砌。
7. 使用高风险概念前，读取 `skills/crossframe/references/concept-cards/README.md` 与对应概念卡。
8. 输出前用 `skills/crossframe/worksheets/concept-fidelity-check.md` 和 `skills/crossframe/worksheets/source-continuity-check.md` 做概念保真与源结构连续性检查。

自动触发时，不要停在本文件。必须继续读取 `skills/crossframe/references/read-routing-map.md`，否则容易只套术语。
