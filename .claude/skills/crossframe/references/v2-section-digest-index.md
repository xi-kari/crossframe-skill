# v2.0 逐节保真摘要索引

本文件配合 `v2-source-spine.md` 使用。它不复刻 2.0 原文，而是给每个源章节一个保真用途摘要、不可误读边界和相邻联读提醒，用于防止拆成 skill 后读少、断章或误读。

## 使用规则

- 本文件只提供逐节摘要和保真提醒，不替代原文。
- 当某节进入判断依据时，必须同时读取其连续联读包。
- 若只读取单概念卡无法保留相邻约束，判断必须降档或补读。
