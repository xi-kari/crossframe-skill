# 五闸工作表

五闸用于防止直接套概念。每一闸都要给出“通过 / 部分通过 / 未通过”，并说明原因。

## 1. 对象闸

- 是否明确诊断对象？
- 是否区分对象与当事人人格？
- 是否说明边界与排除项？
- 未通过时：只能做澄清或轻量观察。

## 2. 证据闸

- 是否区分事实、叙述、解释、推测？
- 是否有重复模式或可复核材料？
- 是否检查反例？
- 是否存在单方叙述限制？
- 未通过时：不能强判断，只能开放断言或低条件观察。

## 3. 尺度闸

- 当前尺度是否明确？
- 跨尺度转移是否必要？
- 是否用高尺度稀释低尺度痛苦？
- 是否用低尺度情绪替代结构判断？
- 未通过时：必须先重设尺度窗口。

## 4. 责任闸

- 谁有行动能力？
- 谁承担成本？
- 谁能改变规则、资源、角色或边界？
- 是否把责任平均化、虚化或转嫁给弱者？
- 未通过时：不能给强行动建议。

## 5. 观测闸

- 这个判断被看见后会改变对象吗？
- 会引发防御、报复、表演性合规或标签化吗？
- 是否需要保护证据、保护弱信号、降低公开性？
- 未通过时：输出应改为开放断言、保护性建议或退出转移。
