## Reading rule

A run traverses all 709 records. Each source route seeds every concept owned by that concept's explicit `primary_source_anchor_id`; `required_neighbor_ids` expands the complete must-read fixed point. `task_signals` and `object_signals` are manually fixed runtime routing terms, not v8 canonical definitions, and never authorize creation or rewriting of canonical concepts. Definitions and allowed inferences must remain exact excerpts supported by an individual registered v8 paragraph anchor, except the explicitly registered multi-line Ω/F definition. Contract bindings cover every recursively derived non-metadata semantic container and leaf pointer, including the closed constraint, and remain bidirectional. Any concept created during a run uses `PROMAX-PROV-*` and cannot be promoted into this canonical file by runtime output.
