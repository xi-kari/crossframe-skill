# /crossframe-explain

使用 CrossFrame 解释一个概念、思想、现象或术语边界。适合“解释虚无主义”“某概念怎么看”“这个说法在 CrossFrame 里是什么意思”。

执行要求：

1. 读取 `skills/crossframe/SKILL.md`。
2. 读取 `skills/crossframe/protocols/concept-explanation-protocol.md`。
3. 读取 `skills/crossframe/references/read-routing-map.md` 与 `skills/crossframe/references/v2-term-fidelity.md`。
4. 如果涉及高风险概念，读取对应 `skills/crossframe/references/concept-cards/`。
5. 输出顺序：推理提纲 -> 人话解释 -> CrossFrame 映射 -> 误读边界 -> 可观察现实信号 -> 可展开诊断入口。
6. 不要写成百科解释，也不要把概念直接用于人格诊断。

用户输入：

`$ARGUMENTS`
