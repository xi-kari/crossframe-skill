# /crossframe

使用 CrossFrame 对用户给出的对象做跨尺度结构诊断。

执行要求：

1. 读取 `skills/crossframe/SKILL.md`。
2. 读取 `skills/crossframe/references/read-routing-map.md`，判断任务类型：诊断、推演、开放断言、低条件行动、高责任审查或概念解释。
3. 如果判断依赖高风险概念，读取对应概念卡和 `skills/crossframe/references/v2-term-fidelity.md`。
4. 先给简短推理提纲：诊断对象、事实边界、尺度窗口、机制候选、判断档位、本次读取的概念、下一步。
5. 再输出面向用户的人话判断。
6. 若证据不足，降低判断档位，不要伪装成强判断。

用户输入：

`$ARGUMENTS`
