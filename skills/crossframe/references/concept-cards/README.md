## 固定阅读顺序

1. 先读最小概念集，确定概念位置。
2. 再读对应概念卡，确认边界、误用和现实信号。
3. 输出时先说人话，概念只作为内部映射或必要补充。

## 禁止

- 不要把概念卡当作输出菜单。
- 不要用概念名替代现实证据。
- 不要因为读了概念卡就提高判断强度；判断强度仍由证据和五闸决定。
- 不要只读取概念名。概念承担判断作用时，必须吸收边界、误用、证据要求和修复动作。
- 深层概念卡默认用于后台保真；普通用户输出只展示现实行为，不展示概念链。
