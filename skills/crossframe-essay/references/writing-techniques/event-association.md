# 因事联想法

- 定义：从一事一物出发，展开相关、相似、对比或因果联想。
- 适用文章类型：思想/概念阐释文章、趋势推演文章、答复体文章。
- 解决的问题：材料太窄，难以进入更高层的意义。
- 操作步骤：先锁定触发事件；写出它唤起的相邻经验；说明联想与中心命题的内在联系。
- 原书操作要点（转述）：原书把这类方法放在“形与神”的关系中理解：先让中心意思立住，再让材料、结构和语言围绕它成形。本技法的关键不是记住名称，而是把“从一事一物出发，展开相关、相似、对比或因果联想”落实到材料选择和段落位置上；执行时按“先锁定触发事件；写出它唤起的相邻经验；说明联想与中心命题的内在联系”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“材料太窄，难以进入更高层的意义”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p27；OCR 对应 PDF_PAGE 036 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：联想过桥句：从一件事引向相似、相反、因果或上位问题。
- 段落前后关系：前接中心命题、典型材料或关键细节；使用它先确立精神焦点。后接事实展开、机制解释或结尾收束，确保所有材料继续围绕主心骨。
- 文章类型微用法：答复体用它抓住读者最在意的一句话；评论体用它确立主心骨；思想文章用它把抽象概念压回可记忆的精神焦点；案例文用它防止材料散开。
- 失败示例（转述）：失败时会把漂亮中心句贴到材料外面，读者只记住口号，却看不见事实、机制和边界。
- 文章类型用法：适用于思想/概念阐释文章、趋势推演文章、答复体文章。优先用于主心骨、首段入口、中心句或结尾收束，让全文有一个稳定的精神焦点。
- 失败形态：联想必须有可说明的关系，不能随意游离；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：多边延伸法、借物喻理法、象征寓意法。
- 误用风险：联想必须有可说明的关系，不能随意游离。
- 输出自检：联想是否让中心更深，而不是让文章跑题？
