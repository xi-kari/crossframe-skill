# 定点换景法

- 定义：固定观察点，按方向、远近、层次转换景物或信息。
- 适用文章类型：公共评论文章、中性分析长文、案例分析文章。
- 解决的问题：需要从一个基点看清多面关系。
- 操作步骤：确定观察基点；按方位或层次展开；每一景都回到基点的意义。
- 原书操作要点（转述）：原书把这类方法看作材料进入表达的秩序：叙述、描写、议论和抒情要随对象变化，而不是平均铺开。本技法的关键不是记住名称，而是把“固定观察点，按方向、远近、层次转换景物或信息”落实到材料选择和段落位置上；执行时按“确定观察基点；按方位或层次展开；每一景都回到基点的意义”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“需要从一个基点看清多面关系”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p113；OCR 对应 PDF_PAGE 122 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：定点换景句：固定观察位置，通过远近、方向或层次变化组织材料。
- 段落前后关系：前接已筛过的事实、场景或用户材料；使用它决定材料怎样被读者看见。后接议论、机制判断或情绪回落，避免表述段变成孤立描写。
- 文章类型微用法：答复体用它把处境说清；公共评论用它交代事实和现场；案例文用它组织人物与时间线；中性分析长文用它降低术语密度。
- 失败示例（转述）：失败时会沉迷描写、节奏或现场感，让材料变成气氛，削弱事实边界和判断档位。
- 文章类型用法：适用于公共评论文章、中性分析长文、案例分析文章。优先用于段落组织、叙述顺序、视角转换或语气节奏，让材料进入可读的表达秩序。
- 失败形态：视线转换要清楚，不能杂乱跳跃；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：点面结合法、随物赋形法、以宾衬主法。
- 误用风险：视线转换要清楚，不能杂乱跳跃。
- 输出自检：读者是否知道本文从哪里看、看向哪里？
