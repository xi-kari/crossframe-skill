# 以动写静法

- 定义：用声音、动作或动态衬托静态和停滞感。
- 适用文章类型：答复体文章、案例叙事文章、思想文章。
- 解决的问题：静态情境难写，容易无味。
- 操作步骤：先建立静的主感受；加入少量动态；让动态反衬静的深度。
- 原书操作要点（转述）：原书把这类方法放在点面、虚实、动静、显隐之间理解：对照和转化要让主体更清楚，而不是遮蔽主体。本技法的关键不是记住名称，而是把“用声音、动作或动态衬托静态和停滞感”落实到材料选择和段落位置上；执行时按“先建立静的主感受；加入少量动态；让动态反衬静的深度”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“静态情境难写，容易无味”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。

- 好句类型：动静互显句：用动作、声音或流程衬出被压住的稳定状态。
- 段落前后关系：前接局部材料或整体背景；使用它处理点面、虚实、多寡、动静关系。后接边界段、机制段或概念上升，保持对照和转化有现实落点。
- 文章类型微用法：中性分析长文用它处理复杂材料；公共评论用它兼顾局部案例和制度面；思想文章用它保留余味；组织复盘用它避免单线归因。

- 文章类型用法：适用于答复体文章、案例叙事文章、思想文章。优先用于复杂关系、点面互照、含蓄表达或边界段，让判断保留层次和余地。
- 失败形态：动要少而准，不能破坏静的主感受；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。

- 误用风险：动要少而准，不能破坏静的主感受。
