# 双桥并立法

- 定义：用双线并行叙述复杂事件，使不同人物、空间或行动互相映照。
- 适用文章类型：公共评论文章、组织复盘/修复文章、案例叙事文章。
- 解决的问题：单线叙述无法呈现多方同时行动。
- 操作步骤：确定两条线；交代时间地点切换；在交汇处提炼机制。
- 原书操作要点（转述）：原书把这类方法看作材料进入表达的秩序：叙述、描写、议论和抒情要随对象变化，而不是平均铺开。本技法的关键不是记住名称，而是把“用双线并行叙述复杂事件，使不同人物、空间或行动互相映照”落实到材料选择和段落位置上；执行时按“确定两条线；交代时间地点切换；在交汇处提炼机制”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“单线叙述无法呈现多方同时行动”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p82；OCR 对应 PDF_PAGE 091 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：双线接桥句：在两条材料线之间设置合流、错位或互证句。
- 段落前后关系：前接已筛过的事实、场景或用户材料；使用它决定材料怎样被读者看见。后接议论、机制判断或情绪回落，避免表述段变成孤立描写。
- 文章类型微用法：答复体用它把处境说清；公共评论用它交代事实和现场；案例文用它组织人物与时间线；中性分析长文用它降低术语密度。
- 失败示例（转述）：失败时会沉迷描写、节奏或现场感，让材料变成气氛，削弱事实边界和判断档位。
- 文章类型用法：适用于公共评论文章、组织复盘/修复文章、案例叙事文章。优先用于段落组织、叙述顺序、视角转换或语气节奏，让材料进入可读的表达秩序。
- 失败形态：双线必须分合有致，不能让读者迷路；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：点面结合法、一石数鸟法、正反相比法。
- 误用风险：双线必须分合有致，不能让读者迷路。
- 输出自检：两条线是否共同说明同一个结构问题？
