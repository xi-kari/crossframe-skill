# 随物赋形法

- 定义：根据材料对象和情感流动自然组织结构，让文章形态顺应对象本身。
- 适用文章类型：案例叙事/案例分析文章、读书互读/吸收文章、中性分析长文。
- 解决的问题：固定结构压扁材料，导致文章机械。
- 操作步骤：观察材料自身秩序；让段落顺序服从信息依赖；保持内在线索。
- 原书操作要点（转述）：原书把这类方法放在“形与神”的关系中理解：先让中心意思立住，再让材料、结构和语言围绕它成形。本技法的关键不是记住名称，而是把“根据材料对象和情感流动自然组织结构，让文章形态顺应对象本身”落实到材料选择和段落位置上；执行时按“观察材料自身秩序；让段落顺序服从信息依赖；保持内在线索”推进。
- 段落动作：在成文时先标出它负责的段落位置，再用它解决“固定结构压扁材料，导致文章机械”；该技法至少要完成一个可见动作，如立题、入场、串联、转折、反驳、回落或收束。
- CrossFrame 适配：只改变表达路径，不改变事实边界、判断档位、来源台账、连续联读包、读态胶囊和源锚点。若技法产生的新句子无法回指底稿，应降为表达转译或删除。
- 原书页码/OCR锚点：目录页码 p61；OCR 对应 PDF_PAGE 070 左右（标题 OCR 可能有少量错字）。本卡依据 OCR 研究笔记和对应页段转述，不复制原书长段。
- 好句类型：随形句：让段落形态跟随材料自身顺序、情感波动或对象结构。
- 段落前后关系：前接中心命题、典型材料或关键细节；使用它先确立精神焦点。后接事实展开、机制解释或结尾收束，确保所有材料继续围绕主心骨。
- 文章类型微用法：答复体用它抓住读者最在意的一句话；评论体用它确立主心骨；思想文章用它把抽象概念压回可记忆的精神焦点；案例文用它防止材料散开。
- 失败示例（转述）：失败时会把漂亮中心句贴到材料外面，读者只记住口号，却看不见事实、机制和边界。
- 文章类型用法：适用于案例叙事/案例分析文章、读书互读/吸收文章、中性分析长文。优先用于主心骨、首段入口、中心句或结尾收束，让全文有一个稳定的精神焦点。
- 失败形态：自由不是无序，必须有可追踪的组织原则；常见失败是为了好看而让技法接管命题、让比喻替代证据，或让段落动作越过胶囊边界。
- 搭配技法：彩线穿珠法、移步换形法、分层论说法。
- 误用风险：自由不是无序，必须有可追踪的组织原则。
- 输出自检：文章结构是否像从对象内部长出来，而不是套模板？
