# 互动打磨模板

```markdown
## 先定方向

我先给三个可能的开头方向。你可以选一个，也可以把两个揉在一起。

### 开头 A：[]
[开头文本]
- 暗含命题：
- 后续会走向：

### 开头 B：[]
[开头文本]
- 暗含命题：
- 后续会走向：

### 开头 C：[]
[开头文本]
- 暗含命题：
- 后续会走向：

## 文章骨架

- 中心命题：
- 第一段负责：
- 第二段负责：
- 第三段负责：
- 边界/反例：
- 结尾余味：

## 下一步

你选定入口后，我只写第一段。写完我们再决定下一段往哪转。
```

## 逐段推进时

每一轮输出：

- 当前段落
- 这个段落在文章里的功能
- 2-3 个下一段方向
- 需要用户决定的问题
